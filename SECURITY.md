# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in The Red Council, please report it responsibly:

1. **GitHub Issues**: Open an issue at [GitHub Issues](https://github.com/sherifkozman/the-red-council/issues) with the label `security`
2. **Direct Contact**: Contact the repository owner directly

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Documentation

For detailed security information, see [docs/security.md](docs/security.md), which covers:
- Secret masking
- Output sanitization
- API hardening
- LLM safety controls
- Threat model
